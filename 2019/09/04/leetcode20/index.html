leetcode20 Valid Parentheses

Given a string containing just the characters ‘(‘, ‘)’, ‘{‘, ‘}’, ‘[‘ and ‘]’, determine if the input string is valid.

An input string is valid if:

Open brackets must be closed by the same type of brackets.
Open brackets must be closed in the correct order.
Note that an empty string is also considered valid.

Example 1:

Input: "()"
Output: true

Example 2:

Input: "()[]{}"
Output: true

Example 3:

Input: "(]"
Output: false

Example 4:

Input: "([)]"
Output: false

Example 5:

Input: "{[]}"
Output: true

Approach 2

  • use char array as stack, create an index to point the position, when traverse the array, if char compair to the char index pointting, these two chars out, index points to previous position, if not, put this char into the position of index points, then index points to the next position.
    Finally, if all chars have their pairs, index should be at the first position of array
  1. converse string to char array
  2. If it is an empty string, return true
  3. Create a new index=1, saving the position of second character of the array
  4. use for cycle to traversing the array and make belowing judgement:
  • if value of index pointing the first char, fill the current char[i] into the char[index], index++, then loop the rest of judgement
  • if previous char of index’s pointting is “(“ and char[i] is “)”, index save the position of previous char (index–)
  • if previous char of index’s pointting is “[“or”{“ and char[i] is “]”or”}”,using ascii table to judge, index saves the position of previous char (index–)
  • else, put the char[i] into char[index] and index++
  1. after the for cycle, judge if index equals 0(back to the first position of array), if yes, then return true, else return false
